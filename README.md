# 江芮菁 Personal Profile Website

這個專案是一個使用 React + Vite 構建的個人形象網站，展示了個人風格、生活分享與音樂推薦。

## 功能特色 (Features)

1.  **個人簡介 (Profile)**: 展示姓名、個性標籤與興趣。
2.  **動態分享 (Posts)**: 支援上傳照片、撰寫標題與內文的發布功能。
3.  **推薦歌單 (Music Playlist)**: 可新增 YouTube 連結與標題的歌單，並自動標示第一首為「最推薦」。
4.  **視覺設計 (Design)**: 採用現代化的 Glassmorphism (毛玻璃) 風格，搭配流暢的動畫效果 (Framer Motion)。

## 如何執行 (How to Run)

1.  安裝依賴套件：
    ```bash
    npm install
    ```

2.  啟動開發伺服器：
    ```bash
    npm run dev
    ```

3.  開啟瀏覽器：
    根據終端機顯示的連結 (通常是 `http://localhost:5173`) 進行預覽。

## 技術堆疊 (Tech Stack)

-   **React**: UI 框架
-   **Vite**: 建置工具
-   **Framer Motion**: 動畫效果
-   **Lucide React**: 圖示庫
-   **CSS Modules / Vanilla CSS**: 自定義樣式
